# eslint-config-a1motion

## Install

```
yarn add --dev eslint-config-a1motion
./node_modules/.bin/eslint-config-a1motion
```

## Configs

## js
```json
{
  "extends": ["a1motion"]
}
```

## react
```json
{
  "extends": ["a1motion/react"]
}
```

## typescript
```json
{
  "extends": ["a1motion/typescript"]
}
```

## typescript with react
```json
{
  "extends": ["a1motion/typescript"]
}
```
